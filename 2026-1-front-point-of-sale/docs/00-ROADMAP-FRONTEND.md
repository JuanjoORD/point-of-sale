# Roadmap Frontend POS Inventario

## Objetivo
Construir interfaz React para operacion diaria de inventario y ventas con permisos por rol.

## Alcance MVP
- Login y rutas protegidas.
- CRUD de catalogos y maestros.
- POS rapido con busqueda por nombre/codigo de barras.
- Cliente con nombre, direccion y NIT en venta.
- Descuento opcional por venta.
- Dashboard ventas del dia.
- Reportes por rango y top productos.

## Orden recomendado
1. Base app y router.
2. Auth y guards de permisos.
3. Catalogos/maestros.
4. POS.
5. Dashboard/reportes.
6. QA UX y ajuste final.
